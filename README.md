# Template - WebPack / Angular

## Dependencies

| Software | Version |
| :------: | :-----: |
| node     | v4.3.0  |
| npm      | 4.4.4   |

## Installation

Clone project
```
git clone https://github.com/ericshortcut/angular-webpack.git
```

Install dependencies
```
cd angular-webpack
npm install
```

Run dynamically
```
npm run start
```

Prepare for production
```
npm run prod
```
